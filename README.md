# 📊 AI-Powered Sales Forecasting Dashboard
### Machine Learning Internship Project – FUTURE_ML_01

🚀 **An End-to-End Machine Learning Project for Retail Sales Forecasting using Python, Time Series Modeling, and Interactive Data Visualization.**

---

## 📌 Project Overview

Retail businesses generate massive amounts of sales data every day. Understanding patterns in this data and predicting future trends is crucial for decision-making, inventory management, marketing strategy, and revenue optimization.

This project presents a **complete machine learning pipeline for sales forecasting**, starting from **data preprocessing and exploratory data analysis to time-series forecasting and dashboard visualization**.

The system leverages statistical and machine learning techniques to predict future sales based on historical retail data. The results are visualized through an **interactive Streamlit dashboard**, enabling users to analyze patterns and future predictions easily.

The primary objective of this project is to demonstrate the **practical application of machine learning in real-world retail analytics**.

---

## 🎯 Objectives of the Project

The main objectives of this project include:

- Understanding real-world retail sales datasets  
- Performing **data cleaning and preprocessing**  
- Conducting **Exploratory Data Analysis (EDA)**  
- Identifying trends and seasonal patterns in sales  
- Building a **time-series forecasting model**  
- Predicting **future sales values**  
- Creating an **interactive visualization dashboard** using Streamlit  

---

## 🧠 Key Features

### 📌 Data Cleaning & Preprocessing

The dataset undergoes several preprocessing steps:

- Handling missing values
- Converting date columns to datetime format
- Sorting records chronologically
- Removing inconsistent data entries
- Preparing the dataset for time-series analysis

Proper preprocessing ensures the dataset becomes **clean, structured, and suitable for modeling**.

---

### 📊 Exploratory Data Analysis (EDA)

EDA is performed to understand the dataset before modeling.

Key analyses include:

- Sales trend visualization
- Time-based aggregation
- Seasonal pattern identification
- Rolling averages and smoothing techniques
- Pattern discovery in historical sales

These steps help identify **seasonality, growth trends, and fluctuations**.

---

### 📈 Time Series Forecasting

A **time-series forecasting model** is used to predict future sales based on historical data.

The model captures:

- Trend
- Seasonality
- Random fluctuations

The system forecasts **future sales values**, enabling businesses to plan inventory and operations efficiently.

---

### 📉 Model Evaluation

Model performance is evaluated using metrics such as:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)

These metrics help measure how accurately the model predicts future sales.

---

### 🖥 Interactive Dashboard

An **interactive dashboard built with Streamlit** allows users to visualize insights.

The dashboard provides:

- Historical sales trend visualization
- Forecast vs actual comparison
- Interactive charts
- Future sales predictions

This makes the solution **easy to use for both technical and non-technical users**.

## ▶️ How to Run the Project

### Step 1: Clone the Repository

```bash
git clone https://github.com/HarshaVardhanM08/FUTURE_ML_01.git
```

### Step 2: Navigate to the Project Folder

```bash
cd FUTURE_ML_01
```

### Step 3: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 4: Run the Streamlit Application

```bash
streamlit run streamlit_app.py
```

The dashboard will automatically open in your browser.
